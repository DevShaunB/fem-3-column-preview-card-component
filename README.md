## Run Locally

Clone the project

```bash
  git clone https://github.com/DevShaunB/fem-3-column-preview-card-component.git
```

Go to the project directory

```bash
  cd fem-3-column-preview-card-component/
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
